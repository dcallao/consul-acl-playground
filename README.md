# consul-acl-playground
Test environment for Consul ACL based on Docker

# Usage
Start up containers.  
```
$ docker-compose up -d
Starting consul-acl-playground_consul-server_1 ... done
Starting consul-acl-playground_consul-agent_1  ... done
```
